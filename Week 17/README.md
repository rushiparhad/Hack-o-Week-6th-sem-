# User Data Export & Analytics 🚀

Secure full-stack system for health metrics, anomaly alerts, and encrypted CSV/PDF exports.

## ✨ What You Get

- 📊 Real-time dashboard with charts + anomaly highlights
- 🔐 AES-256 encrypted metric storage + JWT auth
- 📁 Secure CSV/PDF export with filters (date + severity)
- 🧾 Export activity logs with search + pagination

## ⚡ Quick Start (3 mins)

1. Install deps

```bash
cd backend
npm install
cd ../frontend
npm install
```

2. Create env files

- Copy backend/.env.example -> backend/.env
- Copy frontend/.env.example -> frontend/.env

3. Run backend

```bash
cd backend
npm run dev
```

4. Seed demo data

```bash
cd backend
npm run seed
```

5. Run frontend

```bash
cd frontend
npm run dev
```

6. Open app

- Frontend: http://localhost:5173
- Backend: http://localhost:4000
