# 🚀 Hack-o-Week -- 6th Semester Projects

## 📘 Overview

This repository is a combined monorepo containing all projects developed
during the Hack-o-Week (6th Semester) program. Each week focused on
solving real-world energy analytics and smart campus problems using data
analysis, forecasting, and interactive dashboards.

The objective of this repository is to consolidate all weekly
developments into a single structured workspace demonstrating
progressive learning in:

-   Data Analytics
-   Time-Series Analysis
-   Energy Consumption Monitoring
-   Forecasting Techniques
-   Interactive Dashboard Development

------------------------------------------------------------------------

## 🧩 Repository Structure

Hack-o-Week-6th-sem- │ ├── Week1/ → Peak Hour Electricity Spike Analysis
├── Week2/ → Classroom Usage Forecasting ├── Week3/ → Library Energy
Dashboard └── Week4/ → Real-Time Demand Prediction

Each folder represents one week's independent project with its own
datasets and implementation logic.

------------------------------------------------------------------------

## 📊 Project Descriptions

### 🔹 Week 1 --- Peak Hour Electricity Spikes

Goal: Analyze electricity usage patterns to identify peak consumption
hours.

Key Work Done: - Energy data preprocessing using Pandas - Time-based
aggregation - Peak usage detection - Visualization of electricity spikes

Concepts Used: - Descriptive Analytics - Time-series grouping - Data
visualization

------------------------------------------------------------------------

### 🔹 Week 2 --- Classroom Usage Forecasting

Goal: Predict classroom utilization trends to optimize infrastructure
usage.

Key Work Done: - Temporal data analysis - Usage trend evaluation -
Statistical estimation techniques - Forecast visualization

Concepts Used: - Time-series analysis - Statistical forecasting - Data
aggregation

------------------------------------------------------------------------

### 🔹 Week 3 --- Library Energy Dashboard

Goal: Develop an interactive dashboard to monitor and analyze library
energy consumption.

Key Work Done: - CSV data analysis using Pandas - Interactive dashboard
using Streamlit - Plotly visualizations - Weekly energy usage
estimation - Exam schedule correlation analysis

Technologies: - Python - Streamlit - Plotly - Pandas & NumPy

------------------------------------------------------------------------

### 🔹 Week 4 --- Real-Time Demand Prediction

Goal: Estimate energy demand patterns based on historical consumption
trends.

Key Work Done: - Demand pattern analysis - Trend smoothing techniques -
Statistical estimation models - Forecast-oriented analytics

Concepts Used: - Time-series forecasting - Exponential smoothing
(conceptual implementation) - Demand analysis

## ▶️ Running Projects Locally

1.  Clone repository: git clone
    https://github.com/rushiparhad/Hack-o-Week-6th-sem-.git

2.  Navigate into a specific week: cd Week3

3.  Create/activate virtual environment: python -m venv venv source
    venv/bin/activate

4.  Install dependencies: pip install -r requirements.txt

5.  Run dashboard: python -m streamlit run app.py
